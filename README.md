# 📸 사담: 여행을 사진에 담다.

**사담(여행을 사진에 담다) 서비스**는 사용자들이 직접 찍은 사진의 스팟을 공유하는 서비스입니다.
**인생샷을 공유함**으로써 여행지 추천을 해주고 유익한 정보를 주고받을 수 있습니다:)

<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/8cb5ec2a-9951-40ff-9c0d-fda9090340a0">


***


## 💻 프로젝트 주요 기능

### 🤖 여행 관련 AI 검색 기능
- **GPT 기능을 통해 여행과 관련된 다양한 질문을 할 수 있습니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/7727d8f3-e348-4ace-be7e-1275d792eb00">

### 🗺️ 인증샷 지도 기능
- **현재 게시된 전국의 인증샷 분포를 한눈에 볼 수 있습니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/996751fb-096c-4019-8fb5-f037668ee86d">

### 🏆 인기 인증샷 게시글 랭킹
- **인증샷의 조회순, 인기순으로 인기있는 인증샷 순위를 볼 수 있습니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/909ef124-4b34-4c84-9c52-b1e98cbc9f35">

### 🏖️ 주변 관광지 검색
- **지역별 필터를 적용해 주변 관광지를 검색할 수 있습니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/1ea6b626-d68d-419d-998f-d3bccece52af">

### 🎞️ 인증샷 게시판
- **인증샷이 올라오는 전체 게시글을 확인할 수 있습니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/5ec3b13a-277f-475c-bd2c-d47f52fdda16">

### 🎞️ 게시글 업로드(1단계 - 이미지 업로드)
- **게시글은 총 4단계로 나누어 작성되며, 1단계로 인생샷 이미지를 업로드합니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/dc10d83d-f300-464f-8cd2-1cf07e511761">

### 🎞️ 게시글 업로드(2단계 - 장소 업로드)
- **2단계로 인생샷을 찍은 장소, 상세 정보 등 장소 정보를 입력합니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/d8156aa1-3114-47de-9493-d6f4d136813a">

### 🎞️ 게시글 업로드(3단계 - 날짜 업로드)
- **3단계로 인증샷을 찍은 날짜를 입력합니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/e6a718a6-e21c-4b6d-8305-70d9c30961f9">

### 🎞️ 게시글 업로드(4단계 - 내용 업로드)
- **마지막 단계로 게시글 제목과 내용을 입력하고, 완료 버튼을 눌러 게시글을 업로드합니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/db4e1303-83e9-4a28-b467-1ff2bff475fd">

### ❤️ 내 찜 목록
- **게시글에 ♥️ 버튼을 누르면 내가 찜한 목록을 확인할 수 있습니다.**
<img width="1000" alt="image" src="https://github.com/TRIP-SHOT/.github/assets/76212667/c427cdd1-6ae7-4de3-a711-44ca4eaad20e">
